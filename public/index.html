绑定账号

手动完整 Cookie 复制步骤（兜底方式）：
  1. 用浏览器登录并打开 https://www.xiaohongshu.com/explore。
  2. 按 F12 打开开发者工具，切到 Network / 网络。
  3. 筛选 Fetch/XHR，并在过滤框输入 cookie-name:web_session。
  4. 点击任意小红书作品，选择一个请求，在 Request Headers 中复制完整 Cookie。
  5. 粘贴到上方输入框保存；系统会验证它是否是真登录态，不通过不会入库。
💡 推荐顺序：专用浏览器绑定优先，完整 Cookie 只做兜底：

推荐使用 「打开专用浏览器绑定」。系统会使用项目专属浏览器配置目录打开登录页，您正常登录后再保存并校验真实登录态 Cookie。

* 专用浏览器绑定是推荐授权方式；如绑定失败，再使用手动完整 Cookie。系统每 2 小时会尝试用专用浏览器会话复检/刷新已绑定账号。
